# Milestones

Project milestones define key achievements and progress checkpoints.

## Milestone Structure

Each milestone should include:
- **Goal**: What you want to accomplish
- **Learning Objectives**: Specific skills or knowledge to gain
- **Deliverables**: Concrete artifacts or implementations
- **Success Criteria**: How you'll know the milestone is complete
- **Timeline**: Expected duration

## Template

```markdown
## Milestone N: [Title]

**Goal**: What are you trying to achieve?

**Learning Objectives**:
- Objective 1
- Objective 2

**Deliverables**:
- [ ] Deliverable 1
- [ ] Deliverable 2

**Success Criteria**:
- Criterion 1
- Criterion 2

**Timeline**: Start Date - End Date

**Status**: Not Started / In Progress / Complete
```
